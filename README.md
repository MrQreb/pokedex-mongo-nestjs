# Execute in develop
1. Clone repository.
2. Execute:
```
yarn install
```
3. Must installed Nest CLI  
```
npm -i -g @nestjs/cli
```

4. Get up the data base
```
docker-compose up -d
```

### Stack used
* MongoDB
* Nest 
* Typescirpt
